Medinova is a library for Health & Medical with predefined web elements which helps you to build your own site.

These template is suitable for dental, doctor, care, clinic, corporate, dentist, health, pharmacy, health care, hospital, medical, medicine, fitness, professional, surgeon website.

Medinova template has a fully responsive layout. It fits perfectly on various displays and resolutions from regular desktop screens to tablets, iPads, iPhones and small mobile devices.

This template is built with both Bootstrap 4 & 3.

Note: All images are used for preview purpose only and not included in the final purchase files.
